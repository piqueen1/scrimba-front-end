I'm Learning HTML and CSS

HTML and CSS are the building blocks of the web

There's a shit ton of other really confusing details when you break it down to assets and domain name servers and the rest of it. It's actually sort of magic, but HTML and CSS are the primary legos of the more boring stuff.

HTML
HTML is a markup language which means you include your text and your images, separated into chunks that facilitate their display on the page.
CSS
CSS stands for cascading style sheets, meaning it's a set of rules applied to the CSS that changes its appearance. Different CSS radically changes a webpage.
And More
This could also be spelled JavaScript. JavaScript is the fun part. Neither CSS nor HTML manipulates data in any way but JavaScript primarily does that.
