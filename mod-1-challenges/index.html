What Scrimba has Taught Me -- and how!

The online Scrimba platform has taught me a number of cool things in a variety of ways. Here are some of my favorites.

Games and Projects

The first game I really enjoyed was Pacman. I learned how to model a grid off a huge array spaced like the grid will be, and I learned to programmatically generate divs with createElement and appendChild, two important methods of manipulating the DOM through JavaScript.

Snake is another game with similar building blocks as it also involves traversing a grid using classList.add('class') and classList.remove('class'). It also involved using the modulus operator to test if the snake hit a wall.

Several minor apps were fun to make as well, including a journal app in which I learned about displaying and hiding a DOM node by setting node.style.display = none. In an emoji self-expression app we used myEmojis.push('emoji') to add emojis and myEmojis.pop('emoji') to remove them.

Lessons and Challenges

Because this isn't my first rodeo, I initially skipped to the HTML and CSS Challenges instead of doing the Module 2 lessons in order. Later on I returned to watch every CSS and HTML lesson from the beginning, sometimes while taking a bath.

I learned little interesting things in them, as I did in the JavaSript basics, so I am glad I watched them all (I'm actually still in the process of going back and doing it, whence comes the webpage you are reading.

One fun thing about the lessons is that they are taught by a variety of individuals with interesting accents and intonation patterns. That way if I get sick of one person's voice I can jump to another lesson and another person's voice for a while.

Discord and Github

Two collaborative platforms are also making this adventure more interesting: Discord, which is a community-building platform with direct messaging like Slack, and GitHub, the classic version control website. Discord is motivating because I have new friends who might look at my work and GitHub is very motivating with the little green dots that record daily commits. My current streak is the longest in two years.
